Projects

StarCraft2 Buildinator

The problems:

1. I needed a final project for the Code Louisville Web-Development Program

2. In my time playing StarCraft2, I found the exisiting practice tools for working on builds to be lacking. I wanted to have something I could use at anytime to practice a build that was easily accessible via the web. Current tools, that I found, required users to be in specific pre-made custom modes, or didn't provide an interacive interface with timings.

Features:

  • Built with JavaScript, HTML, and CSS
  • Allows users to manually create builds for any StarCraft race to practice with
  • Uses webworkers to handle hundreds of timings for different objects
  • Allows input of custom JSON data structure to import builds from accompanying project

Github Link: SC2 Buildinator

StarCraft2 Build Extractor

Project Goals:

While building the StarCraft2 Buildinator, I found that the project worked well for simple builds that may have only had a handful of items a user wanted to use. However, I wanted to be able to automate the process a bit and to be able to analyze games from more advanced players.

Features:

  • Built with Python
  • Imports StarCraft2 replay files and pulls data from those replays
  • Saves parsed data as JSON for use in the accompanying webapp

Github Link: SC2 Build Extractor

Playstation 5 Stock-Checker

The Problem:

When the PlayStation 5 first released, it was nearly impossible to get one. Scalpers, bots, and avid fanpeople all wanted to get one. In fact, as of August 2022, it still seems difficult to be able to purchase one at MSRP. My partner is a big PlayStation fan and she really wanted to get one, so I wrote this little project to help.

Features:

  • Built with Python
  • Uses Selenium for webscraping
  • Incorporates the Twilio API for sending automated text messages
  • Uses smtplib for sending automated emails

Github Link: PS5 Stock Checker
